# Client Evidence Required

The following claims are hidden from public rendering or disabled until supporting evidence is supplied.

| Claim | Status | Evidence Required |
|---|---|---|
| 0.6-0.8+ traction coefficient | Pending | Test method, static or dynamic test type, wet/dry condition, tested surface, instrument/equipment, testing laboratory, report date, complete report, and limitations. |
| Up to three years | Pending | Manufacturer-approved technical/commercial documentation defining use cases, exclusions, and maintenance requirements. |
| One to three years between rejuvenations | Pending | Maintenance guide or program documentation with traffic, surface, contamination, and maintenance limitations. |
| Up to 10% insurance-premium reduction | Pending | Insurer documentation, applicable jurisdiction, program conditions, and exact approved wording. |
| $5,000 deductible | Disabled | Client-approved financial basis and exact wording. |
| Five-year price guarantee | Pending | Approved commercial terms, eligibility, exclusions, renewal conditions, territory restrictions, and customer responsibilities. |
| UL-related wording | Disabled | Valid UL certificate or report and approved public wording. |
| ASTM-related wording | Disabled | Complete ASTM test report and approved public wording. |
| Sanitizing | Pending | Technical Data Sheet, Safety Data Sheet, regulatory registration where applicable, and manufacturer-approved marketing material. |
| Antibacterial | Disabled | Regulatory registration and manufacturer-approved wording. |
| Biodegradable | Disabled | Test documentation or regulatory support and approved wording. |
| Non-hazardous | Disabled | Current SDS and jurisdiction-specific regulatory confirmation. |
| pH-neutral | Pending | Current Technical Data Sheet or SDS confirming pH range and wording. |
| Heavy-duty | Pending | Manufacturer-approved marketing material or technical sheet. |
| Insurance reductions | Pending | Insurer documentation, jurisdiction, conditions, and approved wording. |
| WCB-rate reductions | Disabled | Jurisdiction-specific WCB documentation and approved wording. |
| Certification language | Disabled | Valid certificate/report and exact approved public wording. |

Test-result evidence must include the method, static/dynamic test type, wet/dry condition, tested surface, instrument, lab, report date, complete report, and limitations.

Insurance evidence must include insurer documentation, applicable jurisdiction, program conditions, and exact approved wording.

Product-property evidence must include a current Technical Data Sheet, Safety Data Sheet, applicable regulatory registration, and manufacturer-approved marketing material.
